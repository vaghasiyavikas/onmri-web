## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Add your logo.png file to the public directory

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```
